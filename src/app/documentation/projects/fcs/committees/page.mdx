# Committees

## Controls and Dynamics

Primary responsibilities:

- flight control firmware on the Teensy 4.1
- IMU integration and attitude estimation
- PID implementation and tuning
- motor mixing and ESC command generation
- RC input parsing
- log analysis for controller tuning

## Hardware and Fabrication

Primary responsibilities:

- airframe selection and assembly
- motor and propeller selection
- custom brackets or mounts
- mass, center-of-gravity, and inertia measurements
- build log maintenance and parameter sheets

## Electronics and Power

Primary responsibilities:

- ESC selection and configuration
- power distribution design
- battery sizing and power budget verification
- wiring harnesses, connectors, and electrical safety
- motor and ESC characterization

## Simulation and Software

Primary responsibilities:

- quadcopter simulation model development
- pre-tuning controller gains before flight
- ground station and telemetry tooling
- data logging pipeline and analysis scripts
- software architecture documentation

## Recommended deliverables per committee

| Committee | Deliverables |
| --- | --- |
| Controls and Dynamics | Firmware modules, gain tables, estimation notes, log reviews |
| Hardware and Fabrication | CAD or build notes, parameter sheet, assembly log |
| Electronics and Power | Wiring diagram, BOM, power budget, safety checklist |
| Simulation and Software | Simulation notebooks, plots, telemetry tools, analysis scripts |
