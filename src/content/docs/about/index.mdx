## Mission

The Applied Engineering Research Lab exists to:

- Execute ambitious engineering projects
- Develop engineering talent
- Foster interdisciplinary collaboration
- Contribute to open knowledge through publications and open-source work
- Bridge academia and industry

## Vision

We envision a lab where:

- Theory learned in class is immediately applied to projects
- Failures are learning opportunities, not setbacks
- Every member leaves more capable than they arrived
- Our work matters beyond the lab

## Values

### Rigor

- Test thoroughly
- Document completely
- Validate claims
- Learn from mistakes

### Collaboration

- Share knowledge freely
- Help each other succeed
- Cross-train across domains
- Build on each other's work

### Ambition

- Tackle hard problems
- Push technical boundaries
- Learn what is needed
- Ship real results

### Openness

- Open-source code
- Publish results
- Teach others
- Document thoroughly

The current flagship effort is the Flight Control System project: a controls-heavy systems platform for embedded software, sensing, simulation, and real prototype validation.
