3. Why the work matters beyond a single semester

## Current flagship projects

### Flight Control System Project

The FCS Project is a student engineering initiative to design, build, and validate a custom flight control system for a quadcopter test platform.

- Flight computer: Teensy 4.1
- Initial controller: cascaded PID
- Core domains: embedded systems, controls, sensor integration, and simulation
- Long-term direction: LQR, MPC, adaptive control, and richer simulation tooling
